# Media-analyse Framework

Generiek raamwerk voor traceerbare, falsifieerbare mediaanalyse.

## Inhoud

- [`FRAMEWORK.md`](./FRAMEWORK.md) — het generieke raamwerk (principes, lagen, dimensies, werkproces)
- [`toepassingen/CEO-vertrekken.md`](./toepassingen/CEO-vertrekken.md) — sector-specifieke invulling voor CEO-vertrekken in de Belgische pers

## Hoe te gebruiken

1. Lees eerst `FRAMEWORK.md` voor de basisprincipes
2. Kies de relevante toepassing onder `toepassingen/`
3. Volg het analysebestand-template voor elk nieuw artikel
4. Update de aggregatietabel na elke analyse
5. Commit + push na elk artikel (nooit in batch)
