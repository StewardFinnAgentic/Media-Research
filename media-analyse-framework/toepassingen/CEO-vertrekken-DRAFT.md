# Protocol: Mediaberichtgeving over CEO-vertrekken
## DRAFT v0.1 — ter validatie

> Sector-specifieke invulling van het [Media-analyse Framework](../FRAMEWORK.md).  
> **Status:** DRAFT — gebaseerd op beschikbare literatuur (zie bronverantwoording onderaan).  
> Doelpubliek van de analyses: CEO's, communicatiedirecteurs, leden RvB, aandeelhouders.  
> Ze zoeken inzicht en herkenning — geen academisch jargon in de output.

---

## Theoretische grondslag

| Model | Bron | Leestatus | Wat het bijdraagt aan dit protocol |
|-------|------|-----------|-----------------------------------|
| Framing Theory | Entman (1993) | 🔍 Abstract + secundaire bronnen | Vier analytische functies van een tekst: definiëren, oorzaak aanwijzen, oordelen, remedie suggereren |
| Appraisal Framework | Martin & White (2005) | 🔍 Abstract + toepassingsstudies | Drie assen voor taalkundige evaluatie: attitude, zekerheid (engagement), intensiteit (graduation) |
| SCCT | Coombs (2007) | ✅ Volledig gelezen | Organisatorische communicatiestrategieën bij negatieve events: deny / diminish / rebuild / bolstering |
| Retorische IM-profielen | Berkin, Aerts & Roszkowska-Menkes (2025) | ✅ Volledig gelezen | Drie retorische CEO-profielen: Formal-Analytical / Assertive / Defensive — elk met eigen talige kenmerken |
| CEO taalpatronen bij gedwongen vertrek | Akyol & Shabani (2025) | 🔍 Abstract alleen | Gedwongen vertrek = ander taalgebruik dan vrijwillig — te verfijnen zodra PDF beschikbaar |

## Wat we zoeken: vijf analysedimensies

### Dimensie 1 — Positie in de narratieve levenscyclus
*(eigen operationalisering, geïnspireerd op Entman's notie van frame-activering over tijd)*

Een CEO-vertrek genereert doorgaans meerdere artikelen. Elk artikel heeft een positie in die cyclus — dat bepaalt mee welk frame dominant is.

| Code | Moment | Omschrijving |
|------|--------|-------------|
| **L1** | Aankondiging | Eerste bericht — vertrek bekendgemaakt, CEO nog in functie |
| **L2** | Bevestiging | Feitelijk bericht na eerder aangekondigde exit |
| **L3** | Vertrek | Artikel op of rond de effectieve laatste dag |
| **L4** | Terugblik | Evaluatief stuk na vertrek: balans van het mandaat |
| **L5** | Opvolgingsfocus | Nieuwe CEO centraal, voorganger als context |
| **L6** | Erfenis | Maanden later — beslissingen voorganger als referentie |

---

### Dimensie 2 — Artikeltype
*(eigen operationalisering op basis van journalistieke genres)*

| Code | Type | Kenmerken |
|------|------|-----------|
| **A** | Nieuws — gepland | Kort, feitelijk, vertrek was verwacht, weinig duiding |
| **B** | Nieuws — plots | Kort, feitelijk, verrassing zichtbaar, geen opvolger |
| **C** | Achtergrond | Lang, meerdere bronnen, carrière en beslissingen centraal |
| **D** | Commentaar / opinie | Expliciet standpunt journalist of expert |
| **E** | Interview | Betrokkene aan het woord in eigen stem |

---

### Dimensie 3 — Toon
*(gebaseerd op Martin & White 2005 — Appraisal Framework: Attitude + Engagement + Graduation)*

**3a. Attitude** — evalueert de tekst het vertrek of de CEO?

| Score | Label | Wat het betekent |
|-------|-------|-----------------|
| −2 | Sterk kritisch | Expliciet negatief oordeel |
| −1 | Licht kritisch | Voorbehoud, vraagtekens, nuance |
|  0 | Neutraal | Feitelijk, geen evaluatief taalgebruik |
| +1 | Licht positief | Erkentelijk, positieve bewoording |
| +2 | Sterk positief | Uitgesproken lof |

**3b. Zekerheid (Engagement)** — hoe stellig spreekt de tekst zich uit?

| Label | Signaalwoorden |
|-------|---------------|
| Laag — speculatief | "zou kunnen", "naar verluidt", "bronnen zeggen" |
| Middel — voorzichtig | "lijkt", "suggereert", "naar eigen zeggen" |
| Hoog — stellig | bevestigde feiten, geen modaliteit |

**3c. Intensiteit (Graduation)** — hoe sterk is het taalgebruik?

| Label | Signaalwoorden |
|-------|---------------|
| Laag — gedempt | "enigszins", "licht", weinig adjectieven |
| Middel — genormaliseerd | standaard journalistieke toon |
| Hoog — versterkt | "verrassend", "ongezien", "scherp", superlatieven |

> ⚠️ *Elk oordeel op deze assen vereist een citaatanker (zie analysetemplate).*

---

### Dimensie 4 — Reden van vertrek: twee lagen
*(geïnspireerd op Akyol & Shabani 2025 — abstract; en Coombs 2007 — crisis responsibility attribution)*

De officiële verklaring en de journalistieke interpretatie worden **apart gecodeerd**. Divergentie tussen beide is analytisch significatief.

**4a. Officiële verklaring** — wat zegt het bedrijf of de CEO zelf?

| Code | Verklaring |
|------|-----------|
| OV1 | Mandaatseinde / natuurlijk verloop |
| OV2 | Persoonlijke keuze / eigen initiatief |
| OV3 | Strategische heroriëntatie |
| OV4 | Wederzijds akkoord (vaag) |
| OV5 | Geen officiële verklaring |

**4b. Journalistieke interpretatie** — wat suggereert of stelt het artikel?

| Code | Interpretatie |
|------|--------------|
| JI1 | Mandaatseinde / gepland |
| JI2 | Performanceproblemen (resultaten, aandeelhoudersdruk) |
| JI3 | Visionsverschil (strategie, cultuur, richting) |
| JI4 | Afwerving / vertrek voor betere kans elders |
| JI5 | Deontologische kwestie (integriteit, gedrag) |
| JI6 | Politieke of bestuurlijke druk |
| JI7 | Onduidelijk / niet geïnterpreteerd |

> Als OV ≠ JI: expliciet noteren als **divergentie** en beschrijven wat er niet gezegd wordt.

---

### Dimensie 5 — Institutioneel narratief
*(gebaseerd op Coombs 2007 SCCT — deny/diminish/rebuild/bolstering; en Berkin, Aerts 2025 — Assertive/Defensive/Formal-Analytical profielen)*

Het bedrijf communiceert het vertrek actief. Die communicatie is strategisch, niet neutraal.

**5a. Is er een institutionele quote?** (RvB, voorzitter, aandeelhouder, woordvoerder)
→ ja / nee

**5b. Welk retorisch profiel heeft die quote?**
*(Berkin, Aerts 2025)*

| Code | Profiel | Kenmerken |
|------|---------|----------|
| **RP-A** | Assertive | Positieve toon, prestatienadruk, "wij"-taal, toekomstgericht, zelfvertrouwen |
| **RP-D** | Defensive | Cognitief, verklarend, hedging ("zou", "indien"), rationalisering |
| **RP-F** | Formal-Analytical | Afstandelijk, onpersoonlijk, technisch, geen emotionele lading |

**5c. Welke SCCT-strategie volgt de communicatie?**
*(Coombs 2007)*

| Code | Strategie | Typische formulering |
|------|-----------|---------------------|
| **S-R** | Rebuild | "We danken X voor zijn uitzonderlijke bijdrage..." — erkentelijkheid, lof |
| **S-B** | Bolstering | "Onze strategie blijft onverminderd..." — continuïteit, herinnering aan prestaties |
| **S-D** | Diminish | Minimalisering van de impact, "beperkte gevolgen voor..." |
| **S-N** | Geen strategie | Puur feitelijke mededeling, geen narrative sturing |
| **S-X** | Distantiëring | "Met wederzijds akkoord" — geen inhoudelijke toelichting |

> Meerdere codes mogelijk. Noteer de **volgorde** — wat wordt eerst gezegd, wat weggelaten.

## Werkproces

1. Artikel ontvangen → opslaan in `artikelen/`
2. Classificatietabel invullen (snelle scan — 5 min)
3. Observaties (Laag 1) — citaten verankeren
4. Interpretaties (Laag 2) — één per observatie
5. Conclusies (Laag 3) + divergentie noteren indien aanwezig
6. Opslaan in `analyses/`, rij toevoegen aan `aggregatie.md`
7. Committen + pushen
8. Opdrachtgever valideert → feedback → nieuwe commit

**Één artikel per run. Altijd committen voor het volgende.**
